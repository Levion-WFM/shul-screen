# Weekly Schedule Poster

Printable Shabbos schedule poster for Beis Medrash DJackson 21.

## URLs

- Live: `https://shul-screen2.vercel.app/weekly-schedule/`
- Preview (static sample): `https://shul-screen2.vercel.app/weekly-schedule/preview.html`
- API: `https://shul-screen2.vercel.app/api/weekly-schedule` → JSON

The shul-website's Schedules page links here.

## Data flow

- **Shul-specific times** (parsha, candle lighting, mincha, shacharit, mincha
  shabbos, maariv) come from the `shabbos_zmanim` Postgres table — edit in
  the main kiosk's `/zmanim-editor.html`.
- **Astronomical zmanim** (plag, shkia, mincha ketana) are fetched from
  Hebcal at request time — no admin action needed.
- **Shiur title/subtitle + footer note** are baked into `index.html`. Edit
  the HTML to change them.

## Files

- `index.html` — the poster renderer. Fetches `/api/weekly-schedule`, paints
  the `template.png` background, overlays the times.
- `preview.html` — static preview with sample numbers, for design iteration.
- `template.png` — the parchment background (1024 × 1536). Swap this file to
  change the visual.
- `template-blank.png` — same template with the baked-in text regions
  masked out, generated by `mask-template.mjs`.
- `mask-template.mjs` — one-shot script to regenerate `template-blank.png`
  from `template.png`. Coordinates are hand-tuned for 1024 × 1536.

## Replacing the template

1. Drop a new 1024 × 1536 PNG in as `template.png`.
2. If the new template has baked-in text that needs masking, re-tune the
   `MASK_REGIONS` coords in `mask-template.mjs` and run
   `node weekly-schedule/mask-template.mjs`.
3. If the new template has different proportions, also update the
   `background-position` / `top` percentages on the overlay elements in
   `index.html`.
